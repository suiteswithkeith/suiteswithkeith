04

Forms

White field · 1px hairline border · radius 2px · padding 13px 14px. Focus (click in): olive border + 0 0 0 3px rgba(111,116,92,.12) glow.

05

Shape, Elevation & Layout

Corners

Sharp 0 by default. Inputs 2px. Dots & badges 50%.

Elevation

Buttons 0 6px 18px /.12. Cards & popovers 0 20px 40px /.12. Low and soft.

Layout

Container max-width 1180px, centered. Section rhythm 90–100px desktop → 56–64px mobile. Grid gaps 40–50px.

Motion & A11y

Transitions ~0.25s ease. Focus ring 2px olive-deep, 3px offset. Honors prefers-reduced-motion.
